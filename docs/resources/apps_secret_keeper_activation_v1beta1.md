# grafana_apps_secret_keeper_activation_v1beta1 (Resource)

Sets the active keeper for a namespace. Only one keeper can be active at a time; if multiple are applied, the last write wins.

Deleting this resource activates the `system` keeper.

## Example Usage

```terraform
resource "grafana_apps_secret_keeper_activation_v1beta1" "active" {
  metadata {
    uid = grafana_apps_secret_keeper_v1beta1.aws_production.metadata.uid
  }
}
```
