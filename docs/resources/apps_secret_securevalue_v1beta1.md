# grafana_apps_secret_securevalue_v1beta1 (Resource)

Manages a Secrets Management secure value.

Note: the namespace is inferred from the provider stack ID (or org ID for OSS).
Use `grafana_apps_secret_keeper_activation_v1beta1` to select the active keeper; deleting that resource activates the `system` keeper.

`value` is write-only and sensitive. Terraform does not store plaintext in state; updates rotate the value.

## Example Usage

```terraform
resource "grafana_apps_secret_securevalue_v1beta1" "db_password" {
  metadata {
    uid = "db-password"
  }
  spec {
    description = "Database password"
    value       = var.db_password
    decrypters  = ["grafana", "k6"]
  }
}

resource "grafana_apps_secret_securevalue_v1beta1" "external_api_key" {
  metadata {
    uid = "external-api-key"
  }
  spec {
    description = "External API key"
    ref         = "path/to/existing/secret"
    decrypters  = ["grafana"]
  }
}
```
